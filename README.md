# Python Beginner Projects

This repository contains my Python practice programs and mini projects created while learning Python fundamentals. These programs cover basic programming concepts and problem-solving techniques.

## Included Programs

### 1. Electricity Bill Calculator

* Calculates electricity bills based on different unit slabs.

### 2. Shopping Discount System

* Applies discounts according to the shopping amount.

### 3. Leap Year Checker

* Determines whether a given year is a leap year or not.

### 4. Number Guessing Game

* Checks whether the user's guess matches the predefined number.

### 5. Triangle Type Checker

* Identifies Equilateral, Isosceles, or Scalene triangles.

### 6. Student Marks Manager

* Stores student marks in a list.
* Finds highest and lowest marks.
* Calculates total and average marks.
* Counts passed and failed students.

### 7. Shopping List Manager

* Add items to a shopping list.
* Remove items.
* Display all items.
* Exit using a menu-driven system.

### 8. String Analyzer

* Calculates string length.
* Converts text to uppercase and lowercase.
* Counts words and vowels.

### 9. Username Checker

* Validates usernames using length, letters, and digits.

### 10. Email Checker

* Performs basic email validation.

### 11. Text Formatter

* Converts text to uppercase, lowercase, and title case.
* Reverses the order of words.

### 12. Word Statistics

* Counts words and characters.
* Finds the longest and shortest word.

### 13. Text Analyzer Challenge

* Counts words, characters, vowels, and consonants.
* Finds longest and shortest words.

### 14. Contact Book Manager

* Add contacts.
* Search contacts.
* Update phone numbers.
* Delete contacts.
* Display all contacts.
* Count total contacts.

### 15. Student Management System

* Add students and marks.
* Search students.
* Display all student records.
* Update marks using a menu-driven interface.

## Concepts Used

* Variables
* Input and Output
* Conditional Statements
* Loops
* Lists
* Dictionaries
* Strings
* Functions and Built-in Methods
* Menu-Driven Programs

These projects were built as part of my Python learning journey to strengthen programming logic and problem-solving skills.
